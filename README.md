# 文件摘要服务

一个基于FastAPI和Oracle Cloud Infrastructure（OCI）生成式AI的智能文件摘要服务，支持多语言摘要生成和多种文件格式解析。

## 功能特性

### 核心功能
- **多格式文件支持**：PDF、Word (.docx)、Excel (.xlsx)、PowerPoint (.pptx)、文本文件 (.txt, .md)
- **多语言摘要生成**：自动生成中文、英文、日文、韩文四种语言的摘要
- **智能内容提取**：从文档中提取目录结构和核心内容摘要
- **批量处理**：支持按批次批量处理多个文件

## 使用流程

### 1. 文件上传
用户上传一个或多个文件到指定批次，系统将文件保存到本地存储并记录元数据到数据库。

### 2. 摘要生成
页面点击 **生成摘要** 按钮，系统将：
1. 解析文件内容（支持PDF、Word、Excel、PPT等格式）
2. 调用OCI生成式AI模型生成多语言摘要
3. 智能解析AI返回的流式响应
4. 将摘要结果保存到Oracle数据库

### 3. 结果获取
摘要生成完成后，状态将变为 **已完成**，此时点击指定批次可查看生成的文档摘要信息。
